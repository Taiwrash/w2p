This is the landing page for the W2P AI-powered agricultural platform.

## Structure
- **index.html**: Main landing page structure.
- **style.css**: Custom CSS variables, responsive design, and animations.
- **script.js**: Interactivity for smooth scrolling, mobile navigation, and fade-in effects.
- **assets/**: Contains images and resources.

## How to Run

Simply open `index.html` in your browser.

Or, to run a local server:
```bash
npx http-server .
```

## Design Highlights
- **Modern Agritech Aesthetic**: Deep earthy greens combined with neon accents.
- **Responsive**: Fully optimized for mobile (key for smallholder farmers).
- **Performance**: Lightweight, no heavy framework dependencies.
